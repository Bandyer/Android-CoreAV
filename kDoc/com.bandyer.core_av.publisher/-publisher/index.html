Publisher

open class Publisher : CapturerObserver

This class represents a publisher of a stream. A publisher is someone who is in read-write mode on a stream. There can be only one local publisher at a time in the room. The publisher needs a capturer ( audio/video ) to be able to fetch the data to stream.

The first step to use the publisher is create it, by instantiating with a user it refers to.
