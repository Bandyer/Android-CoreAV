Properties

canRecord

 abstract val canRecord: Boolean

Indicates whether the receiver can be recorded, or not. Note: Recording is supported in ROUTED rooms only. true if the receiver can be recorded, false otherwise.

canStartRecorder

 abstract val canStartRecorder: Boolean

Indicates whether the receiver can start the recorder, or not. The recorder can be started only when in routed rooms and in connected state. true if the receiver can start the recorder, false otherwise.

isRecording

 abstract val isRecording: Boolean

Indicates whether the receiver is recording, or not. true if the receiver is recording, false otherwise.
