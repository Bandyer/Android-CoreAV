Room

open class Room : SubscriberObserver, PublisherObserver, SignalingChannelStatusListener, SignalingChannelStreamListener, SignalingChannelMessageListener

This class represents a Bandyer room where the users will join and talk. The first step to use the room is create it by instantiating it with a room token. The room token will be provided by us, beforehand
