putAboveEverything

fun putAboveEverything(putAboveEverything: Boolean): Unit

Method used to put the view above everything

Calling this method multiple times has the same result as calling SurfaceView.setZOrderOnTop

Calling this overrides any previous call to bringToFront.

Parameters

putAboveEverything - true if the view should be above everything, false otherwise
