bringToFront

fun bringToFront(bringToFront: Boolean): Unit

Method used to put the view above others.

Calling this method multiple times has the same result as calling @see SurfaceView.setZOrderMediaOverlay

Calling this overrides any previous call to putAboveEverything.

Parameters

bringToFront - true if the view should be on front, false otherwise
