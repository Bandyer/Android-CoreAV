Publisher

public interface Publisher implements RoomActor, Recordable, Broadcastable
Content copied to clipboard

This class represents a publisher of a stream. A publisher is someone who is in read-write mode on a stream.

The publisher needs a capturer ( audio/video ) to be able to fetch the data to stream.
