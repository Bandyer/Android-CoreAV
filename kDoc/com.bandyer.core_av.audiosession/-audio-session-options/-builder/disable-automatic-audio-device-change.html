disableAutomaticAudioDeviceChange

fun disableAutomaticAudioDeviceChange(): Builder

Call this method to disable automatic audio device change.

By default the AudioSession automatically changes between earpiece -> speaker-phone -> wired-headset -> bluetooth devices in this exact order and priority when connected. Disabling this functionality is means that you want to handle the audio routing manually. Using the function method AudioSession.getInstance().changeAudioOutputDevice(AudioOutputDeviceType).

Return
Builder
