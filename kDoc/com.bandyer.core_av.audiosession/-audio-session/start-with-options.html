startWithOptions

fun startWithOptions(context: AppCompatActivity, options: AudioSessionOptions, listener: AudioSessionListener, proximityListener: ProximitySensorListener?): Unit

Request audio session to initialize with audio session options. Context is requested and never stored. Be sure to follow activity lifecycle and always call dispose on audio session module to prevent activities leaks.

Parameters

context - context used to initialize wired headset and bluetooth broadcast receiver.

options - options to be used in audio session initialization

listener - audio session listener to be notified for hardware output changes.

proximityListener - listener to be notified when the proximity sensor has been triggered
